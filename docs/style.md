# Style Guideline

## Git commit message

-   `[docs]` Update docs.
-   `[test]` Update tests.
-   `[wip]` Work in process.
-   `[func]` Update function.
-   `[feat]` Add a feature.
-   `[conf]` Update config.
-   `[refactor]` Code refactoring.
-   `[perf]` Performance.
-   `[fix]` Fix bugs.
-   `[typo]` Fix typo.
